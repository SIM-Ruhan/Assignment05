1. What is the difference between var, let, and const?
   
answer : In JavaScript, variables can be declared using var, let and const, but their scope and reassignment behave differently. 

var: The more traditional method of declaring variables is with the var keyword.

let: let is used to declare variables with block scope, allowing updates but not re-declaration within the same block.

const: const declares block-scoped variables that cannot be reassigned after their initial assignment.

3.  What is the difference between map(), filter(), and forEach()?
   
answer : map(), filter(), and forEach() are array methods in JavaScript that help us work with arrays of data. However, each of these methods has different uses. 

map(): The map method creates a new array populated with the results of calling a provided function on every element in the calling array. It is used to transform data.


filter(): The filter method creates a new array with all elements that pass the test implemented by the provided function. It is used to include only certain elements from the original array.


forEach(): The forEach method executes a provided function once for each array element. It is typically used to perform side effects rather than to transform the array.

5. What are template literals?
   
answer: Template literals are a modern approach to dealing with strings in JavaScript that involves using backticks instead of single or double quotes. The approach allows JavaScript developers to include variables or expressions within the string using backticks, thus enhancing the creation of dynamic strings. The approach also allows for the creation of multi-line strings without the use of escape characters or concatenation. As such, the use of template literals makes JavaScript string handling flexible and simple compared to other approaches.
